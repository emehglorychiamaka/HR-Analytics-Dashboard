# 📊 HR Analytics Dashboard

## Employee Attrition & Workforce Insights using Power BI

![Dashboard Cover](Images/Cover_Page.png)


# Project Overview

Employee attrition is one of the biggest challenges organizations face. High employee turnover increases recruitment costs, reduces productivity, and affects overall organizational performance.

This project analyzes employee data to identify the major factors influencing employee attrition and workforce behaviour. Using Microsoft Excel for data cleaning and Power BI for data visualization, the dashboard transforms raw HR data into meaningful business insights that support evidence-based decision-making.

The project follows the complete data analytics lifecycle—from data profiling and cleaning to modelling, visualization, business insights, and executive recommendations.


# Business Problem

Employee turnover is expensive.

The Human Resources department needs to understand:

- Which employees are leaving?
- Which departments are most affected?
- Does overtime influence attrition?
- Does salary affect employee retention?
- Which age group leaves most frequently?
- Which job roles have the highest turnover?
- How satisfied are employees before leaving?

Without answering these questions, HR leaders cannot develop effective employee retention strategies.


# Project Objectives

This dashboard was developed to:

- Monitor overall workforce statistics.
- Measure employee attrition.
- Identify departments with high turnover.
- Analyze employee demographics.
- Evaluate employee satisfaction.
- Assess compensation and career growth.
- Provide actionable recommendations for improving employee retention.

# Data Profiling

The dataset was assessed before analysis.

The following issues were identified:

- 10 duplicate employee records
- 57 missing values in YearsWithCurrManager
- Typographical inconsistency in BusinessTravel
- Three unnecessary columns
- Minor formatting inconsistencies


# Data Cleaning

The following cleaning steps were performed:

- Removed duplicate employee records.
- Removed EmployeeCount column.
- Removed StandardHours column.
- Removed Over18 column.
- Corrected BusinessTravel spelling inconsistencies.
- Standardized column formatting.
- Validated unique employee records.
- Created additional analytical columns where required.

# Key Business Insights

The analysis revealed that:

- Research & Development experienced the highest employee attrition.
- Employees aged 26–35 recorded the highest turnover.
- Laboratory Technicians experienced the highest attrition among job roles.
- Employees working overtime were significantly more likely to leave.
- Lower salary groups experienced higher attrition rates.
- Employee satisfaction strongly influenced retention.


# Recommendations

Based on the findings, the following actions are recommended:

- Review overtime policies.
- Strengthen employee engagement initiatives.
- Improve career progression opportunities.
- Review compensation for lower salary bands.
- Increase retention programs for critical job roles.
- Conduct regular employee satisfaction surveys.
